Home-page "From the build plate" images
=======================================

Drop your 4 photos in THIS folder with these exact names:

  print-1.jpg
  print-2.jpg
  print-3.jpg
  print-4.jpg

- .jpg, .png or .webp all work — but if you use a different extension,
  update the matching `src` in app/page.tsx (HOME_GALLERY array at the top).
- Roughly 4:3 / landscape looks best; they're cropped to fill the tile.
- To change the title / material / category shown under each photo,
  edit the HOME_GALLERY array in app/page.tsx.
