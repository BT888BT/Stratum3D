NOW PRINTING — floating print images
=====================================

The "NOW PRINTING" card on the home page cycles through the prints below and
shows each one floating / bobbing. Drop a PNG here for each print using the
EXACT filename shown. Transparent-background PNGs look best (the image floats
over the orange card and casts a soft shadow).

- Recommended size: ~600 x 600 px, square, transparent background.
- If a file is missing, that print falls back to the built-in line drawing —
  nothing breaks, so you can add images one at a time.
- Filenames are lowercase, words separated by hyphens, ".png" extension.
